# Raw Requirements: User Auth System

**Date:** 2026-03-17
**Source:** User Request

## Raw Input
> Hệ thống Quản lý Thành viên (User Auth):
> Backend: Đăng ký, đăng nhập, lưu thông tin người dùng vào Database, băm mật khẩu (bcrypt), cấp JWT.
> Frontend: Form đăng ký/đăng nhập, lưu token, bảo vệ Route.
> Mục tiêu: Kiểm tra khả năng bảo mật và xử lý dữ liệu người dùng.
